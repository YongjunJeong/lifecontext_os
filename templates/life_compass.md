## 현재 시점 판단 필터

> 좋은 선택인지와 별개로, **지금 할 선택인지**를 본다.
> 큰 결정에는 항상 이 섹션을 함께 적용한다.

| 필터 | 질문 | 현재값 |
|---|---|---|
| 방향성 | 내 인생 우선순위·정체성과 맞나? | {{방향성_판단}} |
| 시의성 | 지금 나이·커리어 단계·분기 상황에서 맞나? | {{시의성_판단}} |
| 여력 | 돈·시간·체력·멘탈이 감당 가능한가? | {{여력_판단}} |
| 순서 | 지금 실행인가, 준비만인가, 미룰 일인가? | {{순서_판단}} |
| 리스크 | 지금 하면 잃는 것 vs 미루면 잃는 것은? | {{리스크_판단}} |

### 결론 포맷

```
방향성: 맞음 / 애매 / 아님
시의성: 지금 / 아직 빠름 / 이미 늦음 / 조건부
추천 순서: 지금 실행 / 지금 준비만 / 3개월 뒤 실행 / 1년 뒤 재검토
실행 조건: {{실행_조건}}
하지 말아야 할 버전: {{금지_버전}}
```

### 외부 최신 정보가 필요한 질문

채용 시장, 연봉, 회사 평판, 법·세금·비자, 금리, 산업 트렌드, 국가·업계 관습은 vault만으로 단정하지 않는다.

- 웹검색 가능: 최신 출처 확인 후 판단에 합성
- 웹검색 불가: `최신 확인 필요` 표시 + 확인할 검색어 제안
- 민감 정보는 필요한 만큼만 요약해서 외부 도구에 사용
