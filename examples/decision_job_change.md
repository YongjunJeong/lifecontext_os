# 익명 예시 — 이직 시점 판단

> 이 파일은 실제 개인정보가 아닌 가상 샘플입니다. LifeContext OS가 어떤 식으로 판단을 압축하는지 보여주기 위한 예시입니다.

---

## 입력 컨텍스트

### 현재 상황

| 항목 | 값 |
|---|---|
| 나이 | 31세 |
| 현재 회사 재직 기간 | 1년 9개월 |
| 2년 달성 시점 | 3개월 후 |
| 현금 버퍼 | 생활비 6개월치 |
| 건강 상태 | 수면 부족이 있으나 큰 질병은 없음 |
| 현재 불만 | 성장 속도 둔화, 보상 정체, 의사결정 권한 부족 |
| 현재 강점 | 문제 구조화, 빠른 실행, 고객 커뮤니케이션 |
| 현재 약점 | 흥미가 식으면 급격히 추진력 하락 |
| 이번 분기 목표 | 포트폴리오 정리, 운동 루틴 회복, 이직 시장 탐색 |

### 질문

```text
지금 바로 이직 지원을 시작할까?
방향성 / 시의성 / 여력 / 순서 / 리스크 기준으로 판단해줘.
```

---

## LifeContext OS식 답변 예시

### 결론

**지금은 즉시 퇴사·즉시 이직보다, 3개월 준비 후 2년 달성 직후 적극 지원이 더 좋습니다.**

방향은 맞지만 순서가 조금 빠릅니다.

| 렌즈 | 판단 |
|---|---|
| 방향성 | 맞음. 성장, 자율성, 시장가치 상승 욕구와 연결됩니다. |
| 시의성 | 3개월 대기 권장. 현재 회사 2년 달성 후에는 이력서와 면접 서사가 좋아질 수 있습니다. |
| 여력 | 준비는 가능하지만 즉시 이동은 체력과 포트폴리오가 부족합니다. |
| 순서 | 지금은 준비, 2년 달성 직후 적극 지원. |
| 리스크 | 지금 나가면 잦은 이직 인상이 커질 수 있고, 너무 미루면 성장 정체가 길어집니다. |

### 지금 할 일

1. 이력서와 포트폴리오를 2주 안에 1차 완성합니다.
2. 목표 회사 20곳을 정리하고, 직무별 요구 역량을 표로 만듭니다.
3. 외부 면접 1-2개로 시장 가치를 테스트합니다.
4. 수면과 운동을 회복해 면접 체력을 확보합니다.
5. 2년 달성일 2주 전부터 지원을 시작합니다.

### 최신 확인 필요

이 판단은 개인 컨텍스트 기준입니다. 실제 지원 전에는 아래 정보를 최신으로 확인해야 합니다.

| 확인 항목 | 이유 |
|---|---|
| 목표 직군 채용 수요 | 시장이 얼어 있으면 준비 기간을 늘려야 함 |
| 목표 연봉 밴드 | 협상 기준선 필요 |
| 국가·업계별 재직 기간 관습 | 2년 룰의 강도가 시장마다 다름 |
| 목표 회사 재무·채용 동향 | 채용 공고만 있고 실제 채용이 멈춘 경우가 있음 |

## 업데이트 예시

3개월 뒤 새 정보가 생기면 현재값을 바꿉니다.

```text
이전 값: 재직 1년 9개월, 2년 달성 전
새 값: 재직 2년 달성, 포트폴리오 1차 완성, 면접 2곳 진행
현재 확정값: 적극 지원 가능
정정 이력: 3개월 전에는 준비 권장, 2년 달성 후 실행 단계로 변경
```
